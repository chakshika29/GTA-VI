# 🎮 GTA VI – Cinematic Landing Page Experience

A **high-fidelity, cinematic game landing page** inspired by modern AAA releases.
Built with **React, Tailwind CSS, GSAP, and Vite**, this project focuses on **motion, immersion, and clean UI engineering** rather than static layouts.

> This is **not a clone** of any official site — it’s an original frontend experience designed to showcase animation skills, interaction design, and production-ready structure.

---

## ✨ Features

### 🎬 Cinematic Intro

* Mask-based **SVG intro animation**
* Smooth GSAP timeline transitions
* Seamless handoff from intro to main content

### 🧍 Hero Character Composition

* GTA-style poster framing
* Proper viewport-based positioning (no cropping)
* Subtle mouse-based parallax for depth
* Fully responsive across screen sizes

### 🎮 Interactive Call-To-Action

* Primary **Download Now** CTA
* Secondary **Watch Trailer** modal
* Correct pointer-event handling over animated layers

### 🎥 Trailer Modal

* Fullscreen cinematic modal
* Autoplay + mute best practices
* Clean unmount on close (no audio leaks)
* Background scroll lock for immersion

### 🗺️ World / Location Section

* GTA-style world-building section
* Atmospheric location cards
* Responsive grid layout
* Designed for future map or mission expansion

### 🧠 Thoughtful UX Decisions

* No fake downloads
* No intrusive autoplay sound
* Clear visual hierarchy
* Mobile-first considerations

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/gta-vi-landing.git
cd gta-vi-landing
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

Open your browser at:

```
http://localhost:5173
```
